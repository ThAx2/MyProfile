### 🕵️‍♂️ Offensive Security Focus

- **Web Pentesting:** Especializado en auditorías de aplicaciones web mediante **Ffuf** para fuzzeo de directorios y **Burp Suite Professional** para intercepción y análisis de tráfico.
- **Legacy & Modern Systems:** Capacidad de análisis de vulnerabilidades tanto en entornos modernos (.NET/Java) como en sistemas críticos heredados (**COBOL/VBA**).
- **Red Teaming Lab:** Construcción de entornos controlados en **VirtualBox** con **Kali Linux** y **Debian** para pruebas de penetración y explotación de servicios (DHCP, FTP).
- **Automation for Pentesters:** Creación de scripts personalizados en **Bash**, **Ruby** y **Python** para automatizar la fase de reconocimiento.

### 🚩 Pentesting Methodology

* **Recon:** `Ffuf` para enumeración recursiva y descubrimiento de parámetros ocultos.
* **Analysis:** Mapeo de superficies de ataque con `Burp Suite` y `Nmap`.
* **Exploitation:** Desarrollo de exploits personalizados en `Ruby` y `Bash`.
* **Hardening:** Aseguramiento de servidores **Debian** y **Windows Server** post-auditoría.
